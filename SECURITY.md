# Security Policy

## Cryptography and Algorithms

This project only decrypts Home Assistant backups.  If there's a problem in the
cryptography or algorithm selection itself, then you are advised to [report
upstream to Home Assistant].

## Reporting CLI vulnerabilities

### Supported Versions

Versions currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :x: |
| 0.2.x   | :x: |
| 0.3.x   | :white_check_mark: |

### Responsible disclosure

Report through [GitHub security tab][security].

[security]: https://github.com/samrocketman/home-assistant-decrypt-backup/security
[report upstream to Home Assistant]: https://www.home-assistant.io/security
